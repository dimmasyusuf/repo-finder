## Get Started
1. Clone Repository
```bash
git clone https://github.com/dimmasyusuf/repo-finder.git
```

2. Open Directory
```bash
cd repo-finder
```

3. Install Packages
```bash
npm install
```

4. Start Server
```bash
npm run start-dev
```

## Build for Production

- Build
```bash
npm run build
```
